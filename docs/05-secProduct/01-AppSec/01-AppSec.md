# AppSec (Application Security) 应用程序安全

## 0x00 总结

专注于 Web 应用程序、移动应用程序，围绕软件系统安全，主要就是安全开发实践、安全测试。

简单来说只要是服务于应用安全的产品都可以往这个理念靠，为了和 DevSecOps 线产品区分开，会把传统安全产品分类到该 tag 下。

## 0x01 发展

### 分类

AppSec 主要分为三大类

1. WAF（Web 应用防火墙）
2. 漏洞管理、评估
3. AppSec 测试（SAST、DAST、IAST、SCA）

### 供应方式

1. DevSecOps：DevOps + AppSec，安全乙方的主力区，强调 "安全左移" 这个概念。

2. 监控、可观测：WAF、RASP（已有传统 WAF 厂商引入 RASP 能力的做法）
3. 云原生：比较主力的方向是 CNAPP，区别是提供了更多的定制扫描能力，云厂商主力区，容器镜像扫描、SCA 能力都有往这块靠。
