# ASOC (Application Security Orchestration And Correlation) 应用程序安全编排和关联

## 0x00 总结

ASOC 解决方案从各种 AppSec 来源（如 SAST、DAST 和 IAST 工具）收集数据，并将这些数据整合到单个数据库中。然后，ASOC 解决方案将这些结果关联起来，对关键修复工作进行优先排序。

ASOC 是 AppSec 的一种类别，Gartner 在分类时将 ASOC 和 ASPM 描述的意思非常接近，实际上 ASPM 是包含了 ASOC 的能力，并加上额外的扫描能力，所以划分到 ASPM 篇章下。
